## 💡 料金に影響を与える要因

| 要因 | 方向性 |
| :--- | :--- |
| 都市部（東京・大阪等） | ↑ 高め |
| 地方・地域密着型 | ↓ 低め |
| 専門特化（建設業など特定業種） | ↑ 高め |
| 社労士法人（大規模） | ↑ 高め・交渉可 |
| 顧問契約あり（スポット割引） | ↓ 割引あり |
