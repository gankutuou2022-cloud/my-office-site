## 📢 訴求メッセージの3本柱

### 1. 「守り」のコンプライアンス（法規制対応）
- カスハラ対策の義務化（条例・指針）への完全適合。
- 「安全配慮義務」を怠ることによる訴訟リスクの回避。

### 2. 「攻め」の心理的安全（定着・生産性）
- 従業員が安心して働ける環境こそが、最高の定着策である点。
- ハラスメント対策は損害回避ではなく、「人材投資」であるというポジショニング。

### 3. 「プロ」の実績（信頼性）
- 登壇回数、受講者数、メディア掲載実績などの客観的エビデンス。

## 📐 サイト構成（アレンジ版）

### 1. トップページ（index.html）
- **ヒーロー**: 「その一線が、企業の未来を分ける。」のような鋭いコピー。
- **ニュース**: 登壇予定、最新の法改正情報。
- **実績ギャラリー**: 有名企業や団体での登壇ロゴ、受講者の声。

### 2. 講師プロフィール（about.html）
- 社労士としての専門背景 ＋ 講師としての想い。
- 登壇スタイル（対話型、ワークショップ形式など）。

### 3. セミナーメニュー（seminar.html）
- カスタマーハラスメント（カスハラ）対策研修
- 管理職向けパワハラ防止研修
- 心理的安全性向上ワークショップ

### 4. 登壇・メディア実績（achievements.html）
- 過去の主要な講演実績一覧。
- 雑誌寄稿、テレビ出演、著書。

### 5. お問い合わせ・見積依頼（contact.html）
- 研修・講演の依頼専用フォーム。

## 🏗️ 実装のポイント（config.js の拡張）

ハラスメント版では、標準版の変数を活かしつつ、以下の項目を追加・拡張する。

```javascript
const HARASSMENT_CONFIG = {
  // 追加項目
  seminarStats: {
    counts: "500", // 登壇回数
    trainees: "20,000", // 受講者累計
  },
  keyThemes: [
    "従業員の心理的安全性の確保",
    "安全配慮義務の履行",
    "カスハラ条例・法規制への適合"
  ],
  // ...既存のofficeName等
};
```
