## ⚠️ ギャップ・課題：補足が必要なもの

### 🔴 クリティカル（サイト作成に直接影響）

| 課題 | 内容 | 対処方法 |
| :--- | :--- | :--- |
| **実際のHTMLファイルが存在しない** | 設計図はあるが、実装された `index.html` 等がゼロ | → 今すぐ私が作成する |
| **SEO設定の記述がない** | `<title>`,`<meta description>`, OGPタグの指定なし | → 追加仕様を決めて実装 |
| **画像素材が未定** | `profile-placeholder.jpg`等のプレースホルダー画像未用意 | → AI生成 or Unsplash素材を活用 |
| **サービス料金の具体的な数字** | `pricing.html`の内容が「○○円」のまま | → サンプル料金の相場リサーチが必要 |
