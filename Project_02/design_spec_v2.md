# デザイン仕様書 v2（改訂版）

**作成日**: 2026/03/01
**目的**: 「AI作成っぽい」を脱却し、信頼感・高級感・社労士らしさを出す

> ⚠️ **改訂の背景**
> 旧デザイン案（Archive参照）はリサーチとして有効だったが、
> Font Awesomeアイコン多用・グラデーション背景・ライトブルー主体の構成は
> 「2024年型のテンプレートサイト」に見えるリスクが高い。
> 本書は「本物の事務所サイト」に見せるための改訂版仕様である。

## 🚫 「AI作成っぽさ」を避けるためのNGリスト

| NG要素 | 理由 | 代替案 |
| :--- | :--- | :--- |
| Font Awesomeアイコンをカードごとに配置 | 最もテンプレートらしい見た目 | アイコンは最小限。使うなら線画SVG |
| グラデーションのヒーロー背景 | Wix・AI生成の典型パターン | 単色 or 写真 + 濃いオーバーレイ |
| 「信頼」「安心」「実績」の連呼 | AI生成コピーの典型 | 具体的な数字・専門分野で語る |
| CSSアニメーションの多用 | 安っぽさに直結 | hover時のみ、且つ控えめに |
| 水色 × オレンジの配色 | 2020年代前半の「明るいLP」感 | ネイビー × ゴールドに置換 |

## ✅ 「本物らしさ」を出すためのポイント

### 1. 代表挨拶は「人間が書いた文章」にする
```
❌ AIっぽい例：
「私たちは、中小企業の皆様の労務管理を
 トータルにサポートする専門家集団です。」

✅ 人間らしい例：
「社労士になって10年、気づけば顧問先の
 多くが製造業の現場を持つ会社でした。
 現場を見ながら考える労務管理が、
 私のスタイルです。」
```

### 2. 実績は「具体的な数字」で語る
```
「顧問先 47社」「開業 11年」「助成金申請 延べ 83件」
（テンプレートではサンプル値として記載し、差し替えを促す）
```

### 3. 専門分野を「1つ」絞ってアピール
```
汎用的な表現：「各種労務手続きを幅広くサポート」
専門性の高い表現：「建設業の社会保険適用拡大に
                    特化したサポートを提供しています」
```

### 4. 写真の優先順位
```
① 代表者の顔写真（最優先）
② 事務所内観 or 相談シーン（次点）
③ やむを得ない場合のみ素材写真（日本人・ビジネス系に限定）
```

## 🗂 ページ別デザインポイント

### トップ（index.html）
- ヒーロー：オフホワイト背景 + 左にキャッチコピー + 右に代表写真（斜めクリップ）
- 強みセクション：カードではなく、**番号付きリスト形式**（1. 2. 3.）
- 実績カウンター：数値のみ大きく表示（アイコン不要）

### 事務所概要（about.html）
- 代表挨拶を最上部に（写真＋テキスト横並び）
- プロフィールは「経歴の羅列」ではなく「なぜ社労士になったか」のナラティブ形式

### 業務内容（services.html）
- 各サービスは「何ができるか」ではなく「どんな悩みが解決するか」で書く
- 対象企業規模・業種を明記してターゲットを絞る

### 料金（pricing.html）
- 「〇〇円〜」の表記は必ず「目安」「要お問い合わせ」を併記
- 顧問・給与計算・スポットの3列で見やすく比較
