# プロジェクト ①: 公開作業チェックリスト

公開作業の各段階でミスがないか確認するためのシートです。

## 🟦 1. GitHub 準備フェーズ
- [ ] GitHubアカウントにログインできるか
- [ ] リポジトリの名前は適切か
- [ ] 必要なファイル（index.html等）がすべて揃っているか
- [ ] 誤ってAPIキーやパスワードなどをコードに含めていないか

## 🟨 2. Cloudflare 連携フェーズ
- [ ] Cloudflareで「Pages」タブを選択しているか（※Workersと間違えない）
- [ ] 正しいGitHubリポジトリを選択したか
- [ ] ビルドコマンドが「空欄」になっているか（静的サイトの場合）

## 🟩 3. 公開確認フェーズ
- [ ] 発行された `xxx.pages.dev` のURLで正常に表示されるか
- [ ] スマホで見たときにレイアウトが崩れていないか
- [ ] 全てのリンクボタンが正しく動作するか
- [ ] 画像が表示されているか

## 🟧 4. 運用フェーズ
- [ ] GitHubで修正を行った際、数分以内に反映されるか
- [ ] お問い合わせフォームなどの外部連携が機能しているか
